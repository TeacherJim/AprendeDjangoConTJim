### Visual Studio Code icon

El icono `vscode.svg` es una representación simplificada del logotipo de Visual Studio Code de Microsoft, utilizado únicamente como indicador visual de bloques de código tipo editor. Microsoft permite el uso del logotipo de VS Code para referenciar el producto. Ver: https://code.visualstudio.com/brand
